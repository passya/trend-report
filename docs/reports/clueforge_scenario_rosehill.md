## 1. 시놉시스 (스포일러 포함)

로즈힐 장원의 여주인 **에밀리아 블랙우드**(67세)가 정원 만찬 후 의식불명으로 발견된다. 독극물 중독 의심. 만찬 참석자 6명 모두 자리를 비우지 않았다고 주장하나, 시누이 **빅토리아 블랙우드**(45세)가 범인이다.

**범행 동기:** 에밀리아가 유언장을 변경하여 빅토리아의 상속분을 대폭 축소하려 했음.  
**범행 수단:** 온실에서 재배 중인 디기탈리스(여우장갑)에서 독소를 추출, 와인에 투입.  
**범행 기회:** 만찬 직전 주방에서 와인 병에 독을 넣음. 집사 토마스가 서빙하여 직접 접촉 흔적 없음.

## 2. 세계관 / 장소 설정

### 로즈힐 장원 (Rosehill Manor)

영국 코츠월즈 지방의 19세기 빅토리아 양식 저택. 블랙우드 가문이 3대째 거주. 광활한 장미 정원과 빅토리아 시대 온실이 특징.

### 장소 맵 (6개)

```
┌─────────────────────────────────────────┐
│              로즈힐 장원 1층             │
│                                         │
│  ┌────────┐        ┌────────┐           │
│  │ 서재    │────────│ 식당    │           │
│  │ Study   │        │ Dining │           │
│  └───┬────┘        └───┬────┘           │
│      │                  │               │
│  ┌───┴────┐        ┌───┴────┐           │
│  │ 온실    │        │ 주방    │           │
│  │Greenhouse│       │Kitchen │           │
│  └───┬────┘        └────────┘           │
│      │                                  │
│  ┌───┴────┐        ┌────────┐           │
│  │ 정원    │────────│ 차고    │           │
│  │ Garden  │        │ Garage │           │
│  └────────┘        └────────┘           │
└─────────────────────────────────────────┘
```

| 장소 | 발견 가능 오브젝트 | 핵심 단서 | 레드 헤링 |
|------|------------------|----------|----------|
| 정원 | 꺾인 장미, 떨어진 귀걸이, 흙 발자국 | 흙 발자국 (온실 방향) | 꺾인 장미 |
| 식당 | 좌석 배치도, 깨진 유리잔, 메뉴 카드 | 좌석 배치도 | 깨진 유리잔 |
| 주방 | 와인잔 (잔여물), 빈 약병, 식재료 목록 | 와인잔, 빈 약병 | 식재료 목록 |
| 온실 | 디기탈리스 화분, 원예 도구, 장갑 | 디기탈리스, 장갑 | 원예 도구 |
| 서재 | 유언장 초안, 가문 사진, 보험 증서 | 유언장 초안 | 보험 증서 |
| 차고 | 쓰레기봉투, 자동차 키, 외출 기록부 | 외출 기록부 | 쓰레기봉투 |

## 3. 전체 챕터 구성 (총괄 맵)

이 샘플 시나리오는 엄밀한 "챕터" 구분 없이 **자유 탐색** 방식이지만, 플레이어의 전형적 진행 흐름을 3페이즈로 나눈다.

```
Phase A: 초기 조사 (0~15분)
  → 사건 브리핑, 현장 (정원) 도착
  → 식당·주방 수색 (기본 단서 수집)
  → 첫 NPC 심문 (집사 토마스 or 레오나드)

Phase B: 심화 조사 (15~30분)
  → 온실·서재 수색 (핵심 단서 발견)
  → 복수 NPC 교차 심문
  → 실언 이벤트 유도
  → 증거 보드에서 가설 수립

Phase C: 해결 (30~45분)
  → 결정적 단서 확보 (차고 외출 기록)
  → 가설 정합성 90%↑ 달성
  → 확신 고발 → 해설
```

## 4. 대분기 상세 (YAML)

프로시저럴 시나리오에서 "분기"는 고정 스크립트가 아니라 **플레이어 행동에 따른 동적 상태 변화**다. 아래는 주요 분기점을 정의한다.

```yaml
branch_points:
  # 분기 1: 첫 심문 대상 선택
  - id: "bp_first_interrogation"
    trigger: "플레이어가 첫 번째 NPC에게 말을 건다"
    choices:
      - npc: "토마스 (집사)"
        effect: "서빙 과정 정보 조기 획득 → 주방 단서 활용도 ↑"
        downstream: "식당 좌석 배치 관련 질문 가능"
      - npc: "레오나드 (장남)"
        effect: "가족 관계 정보 조기 획득 → 유언장 단서 방향 유도"
        downstream: "서재 탐색 우선순위 ↑"
      - npc: "클레어 (며느리)"
        effect: "즉시 감정적 반응 → 빅토리아와의 갈등 정보"
        downstream: "빅토리아 경계도 상승 (간접 효과)"

  # 분기 2: 유언장 발견 시점
  - id: "bp_will_discovery"
    trigger: "서재에서 유언장 초안 발견"
    conditions:
      early_discovery: "Phase A에서 발견 (탐색 스킬 보유 시)"
      normal_discovery: "Phase B에서 발견"
      late_discovery: "Phase C에서 발견 (다른 단서로 우회)"
    effect: |
      early: 모든 NPC에게 유언장 관련 질문 가능 → 심문 효율 ↑
      normal: 표준 진행
      late: 다른 단서 조합으로도 해결 가능하나 가설 정합성 도달 느림

  # 분기 3: 빅토리아 대면 전략
  - id: "bp_victoria_confrontation"
    trigger: "빅토리아에게 온실 방문 여부를 질문"
    choices:
      - approach: "직접 추궁"
        effect: "빅토리아 → hostile 상태, 대화 거부"
        recovery: "다른 NPC 통해 간접 증거 수집 필요"
      - approach: "우회적 질문 (꽃 취미 관련)"
        effect: "빅토리아 → anxious 상태, 실언 가능성 ↑"
      - approach: "다른 NPC 증언 먼저 확보 후 대면"
        effect: "반박 불가능한 증거 제시 → 자백 이벤트 (보너스 XP)"

  # 분기 4: 고발 대상
  - id: "bp_accusation"
    trigger: "확신 고발 선택"
    outcomes:
      correct_victoria: 
        result: "사건 해결! 정상 XP + 해설"
        bonus: "동기·수단까지 정확 시 추가 XP"
      wrong_accusation:
        result: "사건 실패. -20 XP"
        option: "재도전 가능 (단서·대화 유지, 새 가설 수립)"
```

## 5. 핵심 페이즈 상세 설계

### Phase A: 초기 조사 (0~15분)

**AI 디렉티브 (LLM 시스템 프롬프트):**

```markdown
## Phase A Game Master Directive

당신은 미스터리 게임의 게임 마스터입니다.
현재 Phase A (초기 조사) 단계입니다.

### 목표
- 플레이어에게 사건의 기본 정보를 자연스럽게 전달
- 정원·식당·주방에서의 수색 결과를 묘사
- 첫 NPC 심문에서 너무 많은 정보를 주지 않되, 조사 방향을 암시

### 톤
- 긴장감 있되 차분한 도입부
- 장소 묘사는 감각적으로 (시각, 후각, 촉각)
- 단서 발견 시 "이것이 중요할 수도 있다"는 뉘앙스

### 제약
- 범인 정보를 직접 노출하지 않음
- NPC는 Phase A에서 최대 "guarded" 상태까지만 진입
- 실언 이벤트는 Phase A에서 발생하지 않음 (최소 3턴 보호)
```

**히든 이벤트:**

```yaml
hidden_event_phase_a:
  - id: "he_gardener_hint"
    trigger: "정원에서 흙 발자국 조사 시"
    condition: "observation_skill >= 1"
    content: "발자국이 온실 방향으로 이어진다. 발 사이즈로 보아 여성의 것."
    effect: "온실 탐색 동기 부여, 용의자 범위 축소 힌트"
```

### Phase B: 심화 조사 (15~30분)

**AI 디렉티브:**

```markdown
## Phase B Game Master Directive

현재 Phase B (심화 조사) 단계입니다.

### 목표
- 핵심 단서 (디기탈리스, 유언장) 발견 유도
- NPC 교차 심문을 통한 모순 발견 촉진
- 실언 이벤트 발생 가능 구간

### 톤
- 긴장감 상승, NPC들의 불안 묘사 강화
- 새 단서 발견 시 "이것으로 퍼즐이 맞춰지기 시작한다" 느낌

### 제약
- 실언은 sentiment_threshold >= 0.7인 NPC에게만
- 빅토리아는 직접 추궁 시 hostile 전환 가능
- 가설 정합성 60%까지 자연스럽게 도달 목표
```

**히든 이벤트:**

```yaml
hidden_event_phase_b:
  - id: "he_gloves_discovery"
    trigger: "온실 장갑 조사 + 약병 분석 완료 후"
    condition: "clue_medicine_bottle.analyzed == true"
    content: "장갑 안쪽에서 미량의 디기탈리스 성분이 검출된다. 이 장갑을 사용한 사람은..."
    effect: "약병과 온실의 직접 연결. 가설 정합성 +15%"

  - id: "he_car_log"
    trigger: "차고 외출 기록부 조사"
    content: "만찬 3일 전, 빅토리아가 '약국'을 방문한 기록. 처방전 없이 어떤 약물을 구매할 수 있었을까?"
    effect: "수단(means) 관련 결정적 단서"
```

### Phase C: 해결 (30~45분)

**AI 디렉티브:**

```markdown
## Phase C Game Master Directive

현재 Phase C (해결) 단계입니다.

### 목표
- 가설 정합성 90% 달성을 위한 마지막 단서 방향 제시
- 확신 고발 시 극적인 해설 연출
- 사건 해결의 카타르시스 극대화

### 톤
- 클라이맥스. 영화의 마지막 반전 장면처럼
- 해설은 셜록 홈즈의 결론 설명처럼 연역적·우아하게
- 범인의 반응은 절제되지만 깊은 감정 표현

### 제약
- 가설 정합성 90% 미만이면 "아직 확실하지 않다"고 안내
- 오판 시 감정적으로 실망하되, 재도전 동기 부여
```

## 6. 멀티 엔딩

프로시저럴 게임이므로 "엔딩"은 스크립트가 아니라 **고발 결과에 따른 변형**이다.

```yaml
endings:
  # 엔딩 A: 완벽한 해결
  perfect_solve:
    condition: "정확한 범인 + 정확한 동기 + 정확한 수단"
    content: |
      빅토리아는 조용히 손을 내려놓았다. "...모두 알고 있었던 거군요."
      경찰이 도착하고, 당신은 증거 파일을 넘기며 정원을 바라본다.
      에밀리아의 장미가 여전히 바람에 흔들리고 있다.
    reward: "+160 XP (풀 보너스)"
    title: "✨ 완벽한 추리"

  # 엔딩 B: 범인 적중, 세부 불완전
  partial_solve:
    condition: "정확한 범인, 동기 or 수단 불일치"
    content: |
      빅토리아가 체포된다. 하지만 재판에서 몇 가지 증거가 부족하다는
      지적이 나온다. 당신의 추리는 맞았지만, 완벽하지는 않았다.
    reward: "+100 XP"
    title: "🔍 미완의 추리"

  # 엔딩 C: 오판
  wrong_accusation:
    condition: "잘못된 범인 지목"
    content: |
      지목한 인물의 알리바이가 확인된다. 진범은 여전히 자유의 몸이다.
      사건 파일이 미제로 분류되지만... 당신은 포기하지 않는다.
    reward: "-20 XP"
    option: "재도전 가능"
    title: "❌ 미제 사건"

  # 엔딩 D: 자백 유도 (히든)
  confession:
    condition: "빅토리아에게 반박 불가 증거 3개 이상 제시 후 대면"
    content: |
      빅토리아의 눈에 눈물이 고인다. "...그래요. 제가 했어요.
      어머니가 모든 것을 클레어에게 주겠다고 했을 때... 저는..."
      자백은 가장 강력한 증거가 된다.
    reward: "+200 XP (히든 보너스)"
    title: "🎭 자백"
```

## 7. 히든 이벤트 목록

| ID | 트리거 | 조건 | 효과 | 발견율 목표 |
|----|--------|------|------|:---------:|
| HE-01 | 정원 흙 발자국 | observation ≥ 1 | 온실 방향 힌트 | 60% |
