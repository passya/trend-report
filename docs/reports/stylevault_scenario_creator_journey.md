## 시놉시스

프리랜서 일러스트레이터 "하은"이 StyleVault를 처음 알게 되어 가입하고, 자신의 스타일로 LoRA를 학습시키고, 첫 라이선스를 판매하고, 수익을 확인하기까지의 전체 여정을 추적합니다. 동시에 인디 게임 개발사 "PixelSprout"의 대표 "정우"가 게임 출시를 위해 StyleVault에서 에셋을 조달하는 여정을 병행합니다.

## 챕터 1: 인지 & 가입 (Day 1)

### 하은 (작가)
하은은 트위터에서 **"내 LoRA가 Civitai에서 무단 배포되고 있다"**는 다른 작가의 포스트를 보고 분노한다. 댓글에서 StyleVault를 발견 — "여기서는 작가 인증 없이 LoRA를 올릴 수 없고, 판매 수익도 작가에게 돌아간다"는 내용.

**핵심 감정:** 분노(무단 도용) → 호기심(해결책?) → 기대(수익화?)

가입 과정:
1. 크리에이터 트랙 선택
2. Artstation 계정 연동 → 자동으로 포트폴리오 120장 크롤링
3. 이메일 인증
4. 시스템이 스타일 분석 결과 표시: "진한 아웃라인 + 수채화 텍스처 + 따뜻한 톤"

**전환 트리거:** "예상 월 수익 $800~$1,200" 표시 → 학습 시작 버튼 클릭

### 정우 (기업)
정우는 인디 RPG "별의 숲"을 개발 중. 캐릭터 일러스트레이터와 계약했으나 예산 부족으로 NPC 80명 전체를 의뢰하기 어렵다. GDC 2026 부스에서 StyleVault 데모를 본 후 가입.

**핵심 감정:** 걱정(예산 초과) → 놀라움(이 가격에?) → 의심(저작권 괜찮나?)

가입 과정:
1. 비즈니스 트랙 선택
2. 사업자 등록번호 입력 + 사용 목적 "인디 게임 NPC 일러스트"
3. Free Trial (7일) 시작

## 챕터 2: 첫 LoRA 생성 (Day 1~2) — 하은

### Phase A: 학습 데이터 준비

하은이 "새 LoRA 만들기"를 클릭. 학습 마법사가 시작된다.

**시스템 안내:**
> "최적의 LoRA를 위해 20~30장의 이미지를 권장합니다. 다양한 구도, 캐릭터, 배경을 포함해주세요."

하은은 포트폴리오에서 25장을 선택하여 업로드. 자동 분석 결과:

```
다양성 점수: 72% (양호)
주요 누락: 남성 캐릭터 (0장), 배경 단독 (0장)
권장: "남성 캐릭터 3~5장, 풍경 2~3장 추가 시 활용 범위 확대"
```

하은은 남성 캐릭터 3장을 추가하여 총 28장으로 학습 시작.

### Phase B: 저작권 선언

각 이미지에 대한 저작권 선언:
- ✅ 28/28장 "본인 창작 확인"
- ✅ 28/28장 "제3자 IP 미포함 확인"
- ✅ 24/28장 Artstation 업로드 이력 매칭 (4장은 미공개 신작 → 수동 확인 대기열)

**히든 이벤트:** 미공개 4장 중 1장이 역이미지 검색에서 다른 작가의 갤러리에서 발견됨 → 시스템이 자동 알림: "이 이미지가 [다른 갤러리 URL]에서도 발견됩니다. 원작자가 맞으시면 해당 플랫폼에서도 업로드해주세요."

하은은 해당 이미지를 제외하고 27장으로 진행.

### Phase C: 학습 & QA

학습 시작 후 18분 만에 완료. QA 리포트:
- 스타일 재현도: 91.5% ✅
- 프롬프트 반응성: 88.2% ✅
- 과적합 위험: 15.1% ✅
- 저작권 충돌: 없음 ✅

데모 이미지 확인 후 만족. "Dohy's Warm Fantasy v1"이라는 이름으로 스토어 등록.

## 챕터 3: 라이선스 설정 & 첫 판매 (Day 2~7) — 하은 & 정우

### 하은의 라이선스 설정

하은은 3단계 티어를 설정:
- Indie: $29/월 (500건, 크레딧 필수)
- Studio: $149/월 (5,000건)
- Exclusive: 협의 ($2,000+)

추가로 "NSFW 불허" + "NFT 민팅 불허" 제한을 활성화.

### 정우의 발견 & 구매

정우가 "RPG NPC 일러스트" 검색 → 하은의 LoRA가 추천 목록 3번째에 표시.

**정우의 의사결정 과정:**
1. 데모 갤러리 확인 — "오, 이 스타일이 우리 게임 분위기와 잘 맞는다"
2. "실시간 데모" 클릭 → "forest elf archer, green cloak" 입력 → 6초 후 결과 확인
3. **"저작권 체인 보기"** 클릭 — 작가 인증 완료, 학습 데이터 27장 전부 원작 확인, QA 통과
4. 가격 확인 — Indie $29/월이면 NPC 80명을 3~4개월에 걸쳐 "별의 숲" 출시 예산 안에서 해결 가능

정우가 Indie 라이선스 구매. **하은에게 첫 구매 알림:**

> 🎉 "IndieDevStudio 'PixelSprout'이 [Dohy's Warm Fantasy v1]을 Indie 티어로 구매했습니다!"

## 챕터 4: 에셋 생성 & 게임 적용 (Week 2~4) — 정우

정우는 Unity에서 StyleVault 플러그인을 설치하고 API 키를 입력.

### 일상적 사용 흐름
```
1. Unity 에디터에서 StyleVault 패널 오픈
2. 프롬프트 입력: "dwarf blacksmith, muscular, leather apron"
3. 옵션 설정: 포즈(standing), 표정(smiling), 크기(1024x1024)
4. "생성" 클릭 → 8초 후 4장 → 가장 맞는 것 선택
5. 에셋이 프로젝트에 자동 임포트 (라이선스 메타데이터 포함)
```

**핵심 발견:** 정우가 "같은 캐릭터, 다른 포즈"를 요청할 때 **캐릭터 시드(Seed)** 기능을 발견. 동일 시드를 사용하면 같은 얼굴/의상의 캐릭터가 다른 행동을 하는 이미지가 생성됨 → NPC 간 일관성 확보.

2주 만에 NPC 60명 생성 완료. 총 비용: Indie 1개월 $29 + 초과 과금 없음.

## 챕터 5: 수익 성장 & 독점 계약 (Month 2~3) — 하은

### 월간 수익 추이
```
1월: $29   (PixelSprout 1건)
2월: $237  (Indie 3건 + Studio 1건)
3월: $890  (Indie 5건 + Studio 2건 + 개인 구매 다수)
```

### 독점 계약 이벤트

3월에 중형 게임 스튜디오 "DragonScale Games"에서 연락:
> "신작 오픈월드 RPG의 전체 NPC 스타일을 귀하의 LoRA로 통일하고 싶습니다. Exclusive 계약 가능한가요?"

하은은 $2,500/월 × 12개월 독점 계약을 수락. 이 순간부터 DragonScale만이 해당 LoRA를 사용할 수 있게 되고, 기존 Indie/Studio 구독자에게는 "독점 계약으로 인해 갱신 불가" 알림이 발송됨.

**감정 곡선:** 기대 → 첫 수익의 기쁨 → 꾸준한 성장의 만족 → 독점 계약의 흥분 → "AI가 내 일자리를 빼앗는 게 아니라 새 수입원을 만들어준다"는 인식 전환

## 챕터 6: 위기 대응 — 무단 LoRA 탐지 (Month 4)

하은에게 알림:
> ⚠️ "외부 플랫폼(Civitai)에서 귀하의 스타일과 유사도 0.94의 LoRA가 발견되었습니다."

하은이 "저작권 충돌 상세" 페이지를 확인:
- 유사 LoRA 제작자: 알 수 없음 (Civitai 익명 업로더)
- 유사도 점수: 0.94 (임계값 0.92 초과)
- 비교 이미지 8장 제공

하은은 **"DMCA 발송"** 버튼을 클릭. StyleVault가 자동으로:
1. Civitai에 DMCA 테이크다운 요청 발송
2. 하은에게 증거 패키지(PDF) 자동 생성 — 원작 업로드 타임스탬프, 스타일 유사도 분석 리포트, 저작권 선언 이력
3. 진행 상태를 대시보드에서 실시간 추적 가능

3일 후 Civitai에서 해당 LoRA 삭제 확인 → 하은에게 성공 알림.
