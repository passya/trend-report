## 메인 페르소나

### 페르소나 1: "하은" — 프리랜서 일러스트레이터

| 항목 | 내용 |
|------|------|
| **나이/성별** | 28세 / 여성 |
| **직업** | 프리랜서 일러스트레이터 (경력 5년) |
| **연 수입** | $25,000~$35,000 (커미션 + 팬아트 마켓) |
| **주 활동 플랫폼** | Artstation, Pixiv, Twitter/X |
| **팔로워** | Artstation 3,200 / Pixiv 8,500 / Twitter 12,000 |
| **주 장르** | 판타지 캐릭터 일러스트 (게임풍) |
| **기술 수준** | 비엔지니어. Photoshop/Clip Studio 전문. AI 도구 경험 없음 |

**동기:**
1. AI에 의한 일자리 위협에 대한 불안 → "빼앗기기 전에 직접 수익화하겠다"
2. 커미션 수입의 한계 (시간당 가치가 정해져 있음) → 패시브 인컴 욕구
3. 자기 스타일이 무단 학습에 사용되는 것에 대한 분노

**성격 키워드:** 창작에 대한 자부심, 기술 거부감 있지만 배우려는 의지, 수익에 민감, 커뮤니티 활동 활발

#### 여정 단계별 행동 디렉티브

```yaml
journey_level_1_awareness:  # 인지 단계
  behavior:
    - "AI 아트 논쟁에 관심이 높고 관련 트윗을 자주 RT"
    - "Civitai에서 자기 스타일 LoRA가 돌아다니는 것을 발견하고 분노"
    - "StyleVault 광고/입소문을 보고 '이거 진짜야?' 의심"
  trigger_to_next: "다른 작가의 성공 사례를 보고 가입 결심"
  
journey_level_2_onboarding:  # 온보딩 단계
  behavior:
    - "가입 후 즉시 포트폴리오 연동 시도"
    - "인증 과정이 번거로울 때 '이것까지 해야 해?' 망설임"
    - "스타일 분석 결과 + 예상 수익을 보고 동기 부여"
  trigger_to_next: "LoRA 학습 버튼 클릭"
  dropout_risk: "인증 과정 중단 (30%)"
  mitigation: "프로그레스바 + '인증 완료 시 프로모션 노출 3배' 인센티브"
  
journey_level_3_creation:  # 학습 & 등록 단계
  behavior:
    - "학습 데이터 선별에 시간을 많이 씀 (완벽주의)"
    - "QA 결과에 민감하게 반응 — 점수가 낮으면 재학습"
    - "라이선스 가격 설정에서 '너무 비싸지 않을까?' 고민"
  trigger_to_next: "스토어 등록 완료"
  dropout_risk: "QA 미통과 시 좌절 (15%)"
  mitigation: "구체적 개선 가이드 + 무료 재학습 1회"

journey_level_4_monetization:  # 수익 실현 단계
  behavior:
    - "대시보드를 하루에 3~5번 확인 (수익 중독)"
    - "첫 구매 알림에 스크린샷 찍어 SNS 공유"
    - "수익이 없는 기간에는 불안감"
  trigger_to_next: "월 수익 $500 달성"
  
journey_level_5_growth:  # 성장 & 확장 단계
  behavior:
    - "추가 LoRA 학습 (스타일 변형 버전)"
    - "독점 계약 제안에 적극적으로 응함"
    - "다른 작가에게 StyleVault 추천"
    - "가격을 올리기 시작"
```

### 페르소나 2: "정우" — 인디 게임 개발자

| 항목 | 내용 |
|------|------|
| **나이/성별** | 32세 / 남성 |
| **직업** | 1인 인디 게임 개발자 (겸 디자이너/프로그래머) |
| **개발 예산** | $5,000 (자비 + 소규모 펀딩) |
| **개발 도구** | Unity, C# |
| **현재 프로젝트** | 2D 스토리 RPG "별의 숲" |
| **아트 능력** | 기본적인 UI 배치 가능, 캐릭터 일러스트는 외주 |
| **기술 수준** | 개발자. Unity 능숙. AI 이미지 생성 경험 있음 (Stable Diffusion 시도) |

**동기:**
1. 예산 한계로 아트 아웃소싱이 어려움 → 저렴한 대안 필요
2. AI 에셋 사용에 관심 있지만 저작권 리스크가 두려움
3. 게임 전체의 아트 스타일 일관성 확보가 최우선 과제

**성격 키워드:** 실용주의, 비용 의식 강함, 기술 친화적, 커뮤니티 활동 적극적 (r/gamedev, r/indiegaming)

#### 여정 단계별 행동 디렉티브

```yaml
journey_level_1_awareness:
  behavior:
    - "r/gamedev에서 AI 아트 관련 토론에 자주 참여"
    - "Stable Diffusion 직접 사용 경험 있으나 저작권 문제로 포기"
    - "GDC/DevCon 부스에서 StyleVault 데모에 관심"
  trigger_to_next: "저작권 체인 설명을 듣고 '이거면 법적으로 안전하겠다' 판단"

journey_level_2_evaluation:
  behavior:
    - "Free Trial로 데모 생성 10건 테스트"
    - "저작권 체인을 꼼꼼히 확인 — 스크린샷 보관"
    - "가격 대비 아웃소싱 비용과 비교 계산"
    - "커뮤니티에서 동료 개발자에게 물어봄"
  trigger_to_next: "데모 결과물이 원하는 수준 + 가격이 예산 내"
  
journey_level_3_integration:
  behavior:
    - "Unity 플러그인 설치 후 즉시 테스트"
    - "캐릭터 시드 기능을 발견하고 흥분"
    - "대량 생성을 시도하며 워크플로우 최적화"
  trigger_to_next: "게임에 필요한 에셋의 50% 생성 완료"

journey_level_4_advocacy:
  behavior:
    - "게임 출시 시 크레딧에 StyleVault + 작가명 명시"
    - "개발 후기를 r/gamedev에 포스팅"
    - "다른 인디 개발자에게 추천"
```

### 페르소나 3: "미나토" — 중형 게임 스튜디오 아트디렉터

| 항목 | 내용 |
|------|------|
| **나이/성별** | 38세 / 남성 |
| **직업** | 아트 디렉터 @ DragonScale Games (직원 50명) |
| **아트팀 규모** | 8명 (일러스트 3, 3D 3, UI 2) |
| **연간 아트 예산** | $200,000 |
| **현재 과제** | 오픈월드 RPG NPC 500명의 일러스트 → 내부 인력만으로는 1년 이상 소요 |
| **기술 수준** | 비기술자. 아트 전문. AI 도구 사용 경험 없음. 법무팀과 자주 소통 |

**동기:**
1. 프로젝트 일정 단축 (NPC 500명을 6개월 → 2개월)
2. 아트 스타일 일관성 유지 (3명 일러스트레이터 간 스타일 차이 문제)
3. 법무팀이 AI 에셋 사용에 대해 "저작권 보증이 있는 경우만" 허용

**성격 키워드:** 품질 지향, 리스크 회피적, 의사결정 시 근거 데이터 필요, 법적 안전 중시

#### 여정 단계별 행동 디렉티브

```yaml
journey_level_1_awareness:
  behavior:
    - "업계 뉴스레터에서 StyleVault 기사를 읽음"
    - "법무팀에 '저작권 체인이라는 게 있으면 안전한가?' 질문"
  trigger_to_next: "법무팀의 '저작권 체인이 검증 가능하면 사용 가능' 답변"

journey_level_2_pilot:
  behavior:
    - "Studio 티어 Free Trial로 팀 내 파일럿 실시"
    - "내부 일러스트레이터의 작업과 LoRA 출력을 비교 평가"
    - "저작권 체인 PDF 리포트를 법무팀에 전달"
    - "생성 품질이 80점 이상이면 도입 승인"
  trigger_to_next: "파일럿 결과 품질 만족 + 법무 승인"

journey_level_3_deployment:
  behavior:
    - "특정 작가의 Exclusive 계약을 추진"
    - "내부 일러스트레이터가 LoRA 출력을 감수(폴리싱) 하는 하이브리드 워크플로우"
    - "월간 저작권 리포트를 법무팀에 제출"
  trigger_to_next: "NPC 500명 완료"
```
